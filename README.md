# Running the Project

## Install dependencies

```bash
npm install

# Running Cypress Tests
Open Cypress UI
npx cypress open

This command opens the Cypress Test Runner interface.

Run tests in headless mode
npx cypress run

Runs all tests directly in the terminal.
```
